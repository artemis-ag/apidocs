GET /sensor/:id - Show Sensor

Get data for a specific sensor in a facility.

curl \
  -u example@agrilyst.com:75AFDB82 \
  https://api.agrilyst.com/api/v2/sensors/123456

Successful Return:

{
  "id": 1,
  "name": "Sensor 1",
  "model": "priva",
  "visible": "true",
  "status": "active"
}

Status Code(s): 200

Failed Return:

Status Code(s): Timeout, Any 4xx or 5xx status code
