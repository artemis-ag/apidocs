GET /sensors - List Sensors

This endpoint displays all available sensors for your organization.

curl \
  -u example@agrilyst.com:75AFDB82 \
  https://api.agrilyst.com/api/v2/sensors

  "status": "active"
}, {
  "id": 2,
  "name": "Sensor 2",
